FZFish Help Guide

Feng Lab Zebrafish Inventory

⚠️ Please do not edit the Google Sheet directly. All changes should be made through this app — editing the sheet manually can break the inventory for everyone.
📊 Google Sheet  ·  📁 Drive Folder  ·  💻 GitHub

🔐 Signing In

Click Sign in with Google using your @bu.edu Google account that has access to the shared inventory Google Drive. Hit allow for the email-sharing prompt (used for changelog tracking) and allow both Google Drive and Google Sheets permissions (used to edit the sheet and upload tank photos). Your session stays active until you sign out. Use Test Mode to explore without saving anything.

➕ Adding a Tank

Click + Add Tank (desktop) or the + button (bottom-right on mobile). You may also scan a new barcode to begin adding.

  • Attempt to scan the barcode with 📷 (The barcode text is in the bottom left of the label!). Fish in the incubator will auto-generate a Tank ID — please put barcodes on as soon as possible.
  • Fill in the line name using A:b;C:D x Z formatting (e.g. dbh:eGFP;CD4:mCherry x CD8:eGFP).
  • Input locations using Rx Sy or Nx Sy where R/N = rack or nursery, S = shelf counting up from the ground.
  • Add any important notes in the notes section.

🧬 Markers

Add markers using the dropdown pickers when editing a tank. Once added to any tank, a marker appears as a filter option across the app.

  • Positive markers (green tags) — transgenes confirmed present
  • Negative markers (red tags) — transgenes confirmed absent
  • Unsorted markers (gray tags) — transgenes not yet confirmed as present or absent; use these for fish whose genotype is still pending

🏷️ Status

Set the tank's current status when adding or editing. Statuses are also used to filter the inventory:

  • Active — Adult fish living on the rack.
  • Nursery — Fish in the nursery system.
  • Incubator — Embryos or early larvae currently in the incubator.
  • Breeding — Currently set up in a breeding cross.
  • Archived — No longer active; kept for record.

🏞️ Pictures

Please add photos at each stage — this includes embryo plates — to make the inventory useful for anyone working on your project. Photos are automatically saved to the shared Google Drive folder. Use the dot under the uploaded photo to adjust the thumbnail crop so the info tape is in frame.

📷 Barcode Scanning

Tap the 📷 icon in the header to scan a tank barcode. The app will jump straight to that tank or start a new tank entry. You can also scan while adding or editing a tank to fill in the Tank ID field automatically.

✏️ Editing, Duplicating & Deleting

Click any tank card to open the detail panel. From there you can:

  • Edit — modify all fields. Changes sync to the Google Sheet immediately.
  • Duplicate — create a new pre-filled tank with the same line, markers, status, and location. Tank ID, photo, and notes are cleared so you can fill them in fresh.
  • Delete from Inventory — permanently removes the tank. This can't be undone.

If you move a tank, update its status and location in the inventory. All changes are also saved to a Changelog tab on the sheet.

🔍 Searching & Filtering

Use the search bar to find tanks. Use the status chips (Active, Nursery, Incubator…) to filter by status. Use the +Markers / −Markers / ?Markers dropdowns to filter by confirmed presence, confirmed absence, or unsorted transgenes. On mobile, tap ⚙ Filters to access all options.

Multi-term search: separate words are AND'd together — fli1 cd4 returns only tanks matching both.

Field prefixes (desktop): use field:value to target a specific field regardless of the Fields selector. Available prefixes:

  • line:fli1 — search line name
  • id:C12345678 — search tank ID
  • notes:spine — search notes
  • loc:R2A — search location
  • marker:eGFP — search any marker (+/−/?)
  • status:nursery — search status

Mix prefixes and plain terms freely: line:fli1 notes:uncertain

☑️ Selection Mode

Tap ☑ Select to enter selection mode. Click individual cards or shift-click to range-select a group. From the selection bar you can:

  • Add to Experiment — bulk-add selected tanks to any existing experiment via the dropdown.
  • Delete — bulk-delete selected tanks from the inventory. This can't be undone.

When inside an experiment, the Select button lets you pick tanks to remove from that experiment (without deleting from inventory).

🧪 Experiments

Use 🧪 ▾ in the header to create or open an experiment. Experiments are named collections of tanks stored as separate tabs in the same Google Sheet — useful for grouping tanks by project or cross.

  • Click + New to create an experiment.
  • Click + Add Tanks to enter selection mode — existing members are pre-selected. Confirm to save the new set.
  • Click ⬤ Group to enter color-assignment mode. Choose a color (green, red, blue, yellow) then click tanks to assign them to that sub-group.
  • Toggle Group by Color in the experiment bar to sort cards by their assigned color, with color-coded section headers.
  • Rename an experiment by clicking ✏️ in the experiment bar.
  • Click Delete to permanently remove the experiment. This can't be undone.
  • Click ✕ Exit to return to the full inventory.
  • You can also add or remove a tank from any experiment directly from its detail drawer.

⟳ Syncing

Click the ⟳ button to pull the latest data from the Google Sheet. The app syncs automatically on sign-in. All edits are written directly to the sheet — no manual export needed.

📲 Install as an App

FZFish can be installed directly on your phone or computer — no App Store needed. It opens full-screen like a native app.

iPhone / iPad (Safari):

  • Open the site in Safari (must be Safari, not Chrome).
  • Tap the Share button (the box with an arrow at the bottom of the screen).
  • Scroll down and tap Add to Home Screen.
  • Tap Add — FZFish will appear on your home screen like any app.

Android (Chrome):

  • Open the site in Chrome.
  • Tap the ⋮ menu (top-right) and tap Add to Home screen.
  • Or look for the Install app banner that appears automatically.

Desktop (Chrome / Edge):

  • Look for the install icon (⊕) in the address bar and click it.
  • Or open the browser menu and choose Install FZFish.
